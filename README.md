# single-cell-RNA-seq-pipeline
This repository demonstrate the single cell RNA-sequencing workflow applied to multiple bladder cancer samples using the `Seurat` R package. The goal is to read raw scRNA data, create Seurat object, perform quality control including the doublet finder (doublet cells) and ambiant RNA removal steps, merge and integrate multiple samples for batch correction and annotate cell clusters. The pipeline is implemented in R using R Markdown compiled in an interactive report below.
